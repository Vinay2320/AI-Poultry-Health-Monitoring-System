 AI-Based Poultry Health Monitoring and Patrolling Robot

📖 Overview

This project presents an AI-powered poultry health monitoring system that combines **Computer Vision**, **Deep Learning**, and **Image Processing** to automatically detect unhealthy chickens in poultry farms.

The system continuously monitors poultry movement using a camera and classifies poultry diseases from fecal images using a Convolutional Neural Network (CNN). It is designed to assist farmers in early disease detection and improve poultry health management.
